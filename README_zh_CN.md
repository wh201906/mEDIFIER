# mEDIFIER

[English](./README.md)  
官方漫步者APP的开源替代版  
更小更轻便  

# 优点
+ 体积小  
  (Android: ~9MB, PC: ~30MB)  
+ 不联网  
+ 减少按键次数，避免磨损  
+ 跨平台  
+ 快速切换蓝牙主机（仅限Android端）  
+ 导出设置到文件
+ 从文件导入设置

# 缺点
+ 仅测试过 W820NB双金标版 和 W200BT Plus  
  (也许能用在 W820NB Plus 和 W820NB 上)  
+ 在Windows端BLE连接几乎不可用  
+ 发送原始指令有可能损坏设备
